## 1. 프로젝트 소개

팀 Speaking potato의 팀원 소개 프로젝트명은 24/7 올타임 레전드입니다. 

저희는 1페이지, 2페이지, 3페이지로 구성하였고,

1페이지는 메인페이지로, 2페이지는 팀원 소개 페이지로, 3페이지는 방명록으로 구성되어 있습니다.

팀원 소개 페이지에서는 저희 팀원들의 성격 및 각오를 보실 수 있고, 방명록 페이지에서는 직접 코멘트를 남길 수 있습니다.
